ðŸ“· ImUpload Overview

ImUpload is a pure JavaScript library to provide an image upload widgets, it is a simple frontend library to help you easily build up your own image related website. It is applied to a variety of image formats, jpg, jpeg, png, gif, that uploaded from user's local computer, it includes two methods to upload image: selected upload and drag and drop upload, solving the image preview showing problem and contains different processing bar styles.

Methods

ðŸ“· Methods Description

Method Name Description
ImUploadGenerator.selectWithPreview() Click an Upload button and select the image from local computer to upload, and the images
will display at the bottom preview frame. Support multiple-time upload, delete image by using the delete button at the
right hand side of the preview. A counter bar indicate how many images have been uploaded,
the number changes synchronically with add/delete images. Two ways to swtich preview images by using side
button or dots. The current display image enlarge when user move mouse over it.
ImUploadGenerator.dragWithPreview() Drag and drop images from local computer to a specific area to upload images and display in preview frame.
The preview has same functions as described above.
ImUploadGenerator.autoSelectUpload() Click an Upload button and select the image and display in a automatic switch preview frame. The default
switch speed is 3 second per image, the speed changing input box is provided to change the speed.
And a delete button can delete the current displayed image.
ImUploadGenerator.changeSpeed() Change the image switch speed in automatic switch preview.
ImUploadGenerator.deleteAuto() Delete button in automatic switch preview.
ImUploadGenerator.fileProgress() A styled progress bar that can be used to show the image uploaded process from user computer
to server side, the rate of progress is displayed at the bar.
