Component Description
Selected Upload Image With Automatic Slide Show Preview The manually switched preview frame changes to auto automatically switched.
Automatic Slide Show Preview Component -- Speed Change Input Box User can change the image switch speed by input the speed in unit second per image,
the default speed is three seconds per image.
Automatic Slide Show Preview Component -- Image Display Style Styled the display image in width 500 and height 300, the values can be changed at
global variables autoPreviewWidth and autoPreviewHeight
Automatic Slide Show Preview Component -- Counter Bar Dislpay the current page number and total images number in (current / total) style.
